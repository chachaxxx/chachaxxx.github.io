简介

  • sass

    Sass是一款强化CSS的辅助工具，它在css语法的基础上增加了变量 (variables)、嵌套 (nested rules)、混合 (mixins)、导入 (inline imports) 等高级功能，通过函数进行颜色值和属性值的运算，也提供控制指令等高级功能和自定义的输出样式；

  • scss

    sass语法进行了改良，Sass 3就变成了Scss(sassy css)。与原来的语法兼容，只是用{}取代了原来的缩进。

使用

变量（$）

变量名使用中划线或下划线都是指向同一变量的。

后定义的变量声明会被忽略，但赋值会被执行，这一点和ES5中var声明变量是一样的。

$highlight-color: #F90;

嵌套

和less一样，用嵌套表示样式父子关系

sass使用隔行缩进，scss使用花括号

嵌套选择器

.container ul {
	// & 父级选择器
    &:after {
        display:block;
        content:"";
        clear:both;
    }
    // ,群组选择器
    /* 表示既h1，又h2 */
    h1, h2 {
      color: red;
    }
}

属性嵌套

.funky {
  font: {
    family: fantasy;
    size: 30em;
    weight: bold;
  }
}
// => 编译为
.funky {
  font-family: fantasy;
  font-size: 30em;
  font-weight: bold; 
}

导入（@import）

和原生css的import有些不一样

/*使用 !default 表示如果本文件中存在变量则不使用本文件值*/
/*App1.scss*/
$border-color:#aaa; //声明变量
@import App2.scss;  //引入另一个SCSS文件
.container {
    border:1px solid $border-color; //使用变量
}
/*App2.scss*/
$border-color:#ccc !default; //声明变量

/*还可以使用嵌套导入*/
.container {
    @import App2.scss;  //引入另一个SCSS文件
    border:1px solid $border-color; //使用变量
}

混合（@mixin 和 @include）

声明

使用@mixin指令声明一个函数，看一下自己的css文件，有重复的代码片段都可以考虑使用混合器将他们提取出来复用。

@mixin border-radius{
    -moz-border-radius: 5px;
    -webkit-border-radius: 5px;
    border-radius: 5px;
    color:red;
}

混合器作用域内的属性都是return的值，除此之外，还可以为函数传参数。

@mixin get-border-radius($border-radius,$color){
    -moz-border-radius: $border-radius;
    -webkit-border-radius: $border-radius;
    border-radius: $border-radius;
    color:$color;
}

也可以设置混合器的默认值。

@mixin get-border-radius($border-radius:5px,$color:red){
    -moz-border-radius: $border-radius;
    -webkit-border-radius: $border-radius;
    border-radius: $border-radius;
    color:$color;
}

使用

.container {
    border:1px solid #aaa;
    @include get-border-radius;         //不传参则为默认值5px
    @include get-border-radius(10px,blue);   //传参
    @include get-border-radius($color:blue,$border-radius:10px);   //具名传参,不用考虑顺序
}

继承 （% 和 @extend）

定义

使用%定义一个被继承的样式，类似静态语言中的抽象类，他本身不起作用，只用于被其他人继承。

%border-style {
  border:1px solid #aaa;
  -moz-border-radius: 5px;
  -webkit-border-radius: 5px;
  border-radius: 5px;
}

使用

.container {
	@extend %border-style;
}
.container1 {   //继承另一个选择器
	@extend .container;
}

常用内置方法

for循环

@for $i from 1 to 3 {
    .div#{$i}{
       height: $i*20px;
    }
}

列表循环

$yanse: red blue black;
@each $i in $yanse {
    .div#{$i}{
      color: $i;
    }

}

while循环

$gao: 1;
@while $gao<4 {
    .div#{$gao}{
        height: $gao*10px;
    }
   $gao : $gao+1;
}

自定义函数

@function add ($bian)
{
    @return $bian+10px;
}

div{
    width: add(5px);
}
