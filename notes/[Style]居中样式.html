简要总结

水平居中较为简单, 共提供了8种方法, 一般情况下 text-align:center,marin:0 auto; 足矣

  • ① text-align:center;

  • ② margin:0 auto;

  • ③ width:fit-content;

  • ④ flex

  • ⑤ 盒模型

  • ⑥ transform

  • 绝对定位

垂直居中, 共提供了8种方法.

  • ① 单行文本, line-height
  • ② 行内块级元素, 使用 display: inline-block, vertical-align: middle; 加上伪元素辅助实现
  • ③ vertical-align
  • ④ flex
  • ⑤ 盒模型
  • ⑥ transform
  • 绝对定位

我们发现, flex, 盒模型, transform, 绝对定位, 这几种方法同时适用于水平居中和垂直居中.

中心思想：

父级有宽度高度的可以用绝对定位等方向硬实现，在自适应上兼容性比较差

父级没有这些的时候可以利用一些css属性，伪元素还有 flex、grid布局实现

水平居中

  • 行内元素：父元素 text-align:center

  • 块级元素：

    • 子元素 margin : 0 auto
    • Flex布局： 父元素 justify-content: center;
    • transform : (绝对定位的一种)
    .son{
    position:absolute;
    left:50%;
    transform:translate(-50%,0);
}
    • 绝对定位 :
    .parent {
    position: relative; // 父级为相对定位
    height: 200px;
}
.son{
    position:absolute;
    width:固定;
    left:0;
    right:0;
    margin:0 auto;
}

.son{
    position:absolute;
    width:固定;
    left:50%;
    margin-left:-0.5宽度;
}

垂直居中

  • 行内元素：父元素有明确高度，子元素 line-height = height

  • 块级元素：

    • 行内块级+vertical-align+伪元素
    .parent::after, .son{
    display:inline-block;
    vertical-align:middle;
}
.parent::after{
    content:'';
    height:100%;
}
    • flex：
    display: flex;
align-items: center;
    • 绝对定位：
    .parent {
    position: relative;
    height: 200px;
}
// 知道大小
.child {
    width: 80px;
    height: 40px;
    position: absolute;
    left: 50%;
    top: 50%;
    margin-top: -20px;
    margin-left: -40px;
}
// 不知道大小
.child {
    width: 80px;
    height: 40px;
    position: absolute;
    left: 50%;
    top: 50%;
    transform: translate(-50%, -50%); // 移动中心对齐
}
    • 伪元素：:after伪类+content内容生成经典应用举例
