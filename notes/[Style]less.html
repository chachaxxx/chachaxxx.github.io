CSS的不足

CSS是一门非程序式语言，没有变量、函数、SCOPE(作用域）等概念。

  • 冗余度大
  • 重复使用的值不易统一维护、不易拓展
  • 没有计算能力

Less简介

一门CSS扩展语言，也称为CSS预处理器。

在CSS的语法基础之上，引入了变量，Mixin (混入），运算以及函数等功能。

变量、嵌套、混入、操作符、函数

Less的使用

变量@

@key:value; // @为前缀，无特殊字符，无数字开头，大小写敏感，结尾分号
@conversion-1: 5cm + 10mm; // 可以运算

映射#

#colors() { // 定义一组映射
    theme: #0094ff;
    success: #93dd00;
}

转义（使用变量）

.a{
    width:@key; // 使用@key使用
}
.status {
    color: #colors[theme]; // 映射的使用#

    &.success {
        color: #colors[success];
    }
}

混合（.a()）

一种复用代码的手段。

// 一个朴实无华的类名
.bordered {
    border-top: solid 1px #000;
}

#menu a {
    color: #000;
    // 将类名 bordered 中的代码混入,不要忘记结尾的分号
    .bordered(); // 或者写成.bordered;  => 将具有bordered的样式border-top
}

作用域

限从最近的作用域下读取变量，换句话说，如果有两个同名变量，那么谁和使用行近谁生效。

嵌套

和scss不一样没有属性嵌套

demo {
    margin: 0 auto;

    .box {
        display: flex;

        .item {
            color: #000;
        }
    }
}
//=>编译后实际上
.demo .box .item {
  color: #000;
}

替代父元素&

    .item {
        color: #000;
    }

    .item.active {
        color: skyblue;
    }
// =>简写为
 .item {
   color: #000;

// "&" 代替了父级类名 ".item"
   &.active {
       color: skyblue;
     }
}
